# Система складского учёта

Программа для автоматизации учёта товарных запасов на складе.

## Функционал
- Добавление новых товаров
- Просмотр списка всех товаров
- Поступление товара (увеличение количества)
- Списание товара (уменьшение количества) с контролем остатка
- Удаление товара из базы
- Поиск товара по названию

## Запуск
```bash
python warehouse.py
